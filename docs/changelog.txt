Phase I

Core Component System:
- Implemented base Component class with lifecycle methods and event handling
- Added Container class for component hierarchy with layout management
- Created VBox and HBox specialized containers for vertical and horizontal layouts
- Implemented property-based API for a more Pythonic interface

Event System:
- Added EventEmitter base class for event handling
- Created EventBus singleton for application-wide events
- Implemented event registration, emission, and propagation

State Management:
- Added State class for component state with change notifications
- Created StateManager singleton for application-wide state

Uniqueness Constraints:
- Enforced unique component names within the same parent
- Enforced unique component IDs across the entire component tree

Testing:
- Added comprehensive test suite for all components and features
